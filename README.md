# Real Estate Consultancy Next.js Template

A reusable Next.js (App Router) website template tailored for Indian real estate consultancy businesses.

Key Features:
- Next.js App Router (JavaScript)
- Tailwind CSS, mobile-first responsive design
- Centralized data (replace /data/*.js)
- SEO-friendly pages with dynamic metadata
- Local blog using JSON
- YouTube embed support
- Sitemap & robots
- Basic lead capture (client-side) with optional local API stub

How to customize:
1. Edit `/data/siteConfig.js` with client details.
2. Replace images in `/public`.
3. Edit services, projects, blogs, videos in `/data`.

Run locally:
- npm install
- npm run dev
